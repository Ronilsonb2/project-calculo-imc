## Quer conferir qual seu IMC? Aproveite e teste meu [Calculador de IMC](https://ronilsonb2.github.io/project-calculo-imc/) 😀
<br/>

Este e um sistema web para Calcular o IMC e retornar o índice. 

Basta adicionar sua altura e peso que o sistema automaticamente fará o cálculo e retornara seu índice.
